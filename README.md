# news_sentimental_classifier
뉴스기사 감성분류 및 KOSPI 연관성 분석


## 프로젝트 소개
KLUE-Bert 모델기반 증시뉴스 긍/부정 감성분류.
뉴스 긍/부정 분류 결과와 KOSPI 지수와의 연관성 분석.
